# 🌐 พื้นฐาน Web Dev ที่ต้องรู้ก่อนใช้ Agentic Coding

คอร์สสำหรับผู้เริ่มต้นที่อยากใช้ AI ช่วยเขียนโค้ด แต่ยังขาดพื้นฐาน Web Development ที่จำเป็น

## 🎯 เรียนจบแล้วจะได้อะไร

- เข้าใจหลักการทำงานของเว็บไซต์ตั้งแต่ต้นจนจบ
- เขียน HTML, CSS, JavaScript พื้นฐานได้
- ใช้เครื่องมือนักพัฒนา (DevTools, Terminal, Git) เป็น
- พร้อมใช้ AI / Agentic Coding อย่างมีประสิทธิภาพ

## 📚 เนื้อหาทั้งหมด 10 EP

| EP | หัวข้อ | เนื้อหา |
|----|--------|---------|
| 01 | เว็บทำงานอย่างไร | Internet, Client-Server, HTTP, URL, Browser |
| 02 | HTML พื้นฐาน | โครงสร้าง, Tags, Semantic HTML, Forms |
| 03 | CSS พื้นฐาน | Selectors, Box Model, Flexbox, Responsive |
| 04 | JavaScript พื้นฐาน | Variables, Functions, DOM, Events |
| 05 | เครื่องมือนักพัฒนา | Chrome DevTools, Console, Network, Debugging |
| 06 | Terminal & Command Line | คำสั่งพื้นฐาน, Node.js, npm, Scripts |
| 07 | Git & Version Control | Commits, Branches, GitHub, Collaboration |
| 08 | โครงสร้างโปรเจกต์ | File Organization, Config, Dependencies |
| 09 | API & การดึงข้อมูล | REST, JSON, Fetch API, Error Handling |
| 10 | เริ่มต้น Agentic Coding | AI Tools, Prompting, Workflow, Best Practices |

## 🖥️ วิธีใช้ Slides

1. เปิดไฟล์ `slide.html` ในแต่ละ EP ด้วย Browser
2. ใช้ปุ่ม **←** **→** หรือคลิกปุ่มนำทางเพื่อเปลี่ยนสไลด์
3. กด **F** เพื่อเข้าโหมดเต็มจอ
4. กด **Esc** เพื่อออกจากโหมดเต็มจอ

## 👤 กลุ่มเป้าหมาย

- ผู้เริ่มต้นที่ไม่มีพื้นฐานเขียนโค้ด
- คนที่อยากใช้ AI ช่วยสร้างเว็บไซต์/แอป
- นักศึกษาที่สนใจ Web Development
- ผู้ที่ต้องการทบทวนพื้นฐานก่อนใช้เครื่องมือ AI
